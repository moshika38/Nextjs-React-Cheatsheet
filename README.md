# Next.js & React Cheatsheet Repository

This repository serves as a comprehensive cheatsheet for essential concepts, patterns, and integrations in Next.js and React. It is organized into isolated folders, allowing you to quickly explore specific topics. Each folder contains its own detailed README and standalone code examples.

---

## 📂 Repository Structure

The core modules included in this project:

* **`api/`** – Next.js API Routes and backend integration examples.
* **`forms/`** – Form handling, validations, and state management in React.
* **`hooks/`** – Practical examples of built-in React Hooks and custom hooks.
* **`layout/`** – UI layouts, responsive scaffolding, and reusable component structures.
* **`Middleware/`** – Authentication, routing, and request filtering using Next.js Middleware.
* **`Next | Prisma | Sqlite/`** – Database management using Prisma ORM with SQLite in a Next.js environment.
* **`Next | Prisma | Supabase/`** – Connecting a cloud-hosted Supabase (PostgreSQL) database via Prisma.
* **`Portal | Pop-up Window/`** – Creating clean modals and overlays using React Portals.
* **`Routing/`** – Dynamic routing, nested routes, and navigation via the Next.js App Router.
* **`RTK | State Management/`** – Global state management patterns using Redux Toolkit.
